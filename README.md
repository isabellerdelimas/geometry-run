# Geometry Run

Geometry Run is a one-button web game where your geometric jumper runs to the right automatically. Your job is to time each jump, land on platforms, avoid spikes, and clear harder levels to unlock new avatars. It now starts with a countdown, uses an original built-in synth soundtrack after your first input, speeds up as you survive, and has longer neon obstacle runs.

## How to play

1. Open `index.html` in a browser.
2. Wait for the **3, 2, 1** countdown, then press **Space** to jump.
3. Tap **Space** for a short jump, hold it a little longer for a medium jump, and keep holding for the highest jump.
4. Land on platforms and avoid the pink spikes.
5. In later levels, glowing blue jump pads launch you automatically.
6. Clear each level to unlock a new jumper avatar.

## Controls

- **Space**: jump / continue to the next level after winning
- **Restart button**: replay the current level
- **Jump button**: touch-friendly jump control

## Run locally

You can open `index.html` directly, or start a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000/` in your browser.

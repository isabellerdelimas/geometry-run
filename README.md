# Geometry Run

Geometry Run is a one-button web game where your geometric jumper runs to the right automatically. Your job is to time each jump, land on platforms, avoid spikes, and clear harder levels to unlock new avatars.

## How to play

1. Open `index.html` in a browser.
2. Press **Space** to jump.
3. Hold **Space** to jump again automatically each time you land.
4. Land on platforms and avoid the pink spikes.
5. In later levels, glowing blue jump pads launch you automatically.
6. Clear each level to unlock a new jumper avatar.

## Controls

- **Space**: jump / continue to the next level after winning
- **Restart button**: replay the current level
- **Jump button**: touch-friendly jump control

## Run locally

You can open `index.html` directly, or start a small local server:

```bash
python3 -m http.server 8000
```

Then visit `http://127.0.0.1:8000/` in your browser.
